MyPkg.Simpson — Method
function Simpson(f, a, b)

Calculate the integral 'f' function from 'x' to 'y' by Simpson formula

Simpson(f, a, b)=(b-a)/6 * (f(a) + 4*f((a+b)/2) + f(b))

Examples

julia> Simpson(2*x, 1, 2)
3
